5.how to use the project
Create Your First Repository
bash
# Local approach
mkdir my-project
cd my-project
git init
git add .
git commit -m "Initial commit"
# GitHub approach
# 1. Create repo on GitHub.com
# 2. Clone it locally
git clone https://github.com/username/repository.git
